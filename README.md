Projeto desenvolvido em Java com o objetivo de praticar o tratamento de exceções utilizando try, catch e exceções personalizadas.

Objetivo:
Demonstrar como tratar erros durante a execução de um programa, garantindo que o sistema continue funcionando corretamente mesmo quando ocorre uma situação inesperada.

Conceitos aplicados:

Tratamento de exceções (try, catch, finally)

Lançamento de exceções (throw)

Criação de exceções personalizadas

Validação de regras de negócio

Organização de código em camada

Estrutura do projeto

O sistema é composto por classes responsáveis por representar regras e possíveis erros do sistema.
